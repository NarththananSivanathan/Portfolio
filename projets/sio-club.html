Projet de Gestion des Étudiants, Groupes et Événements en Java EE

Introduction :

  • Création d'un site web Java EE pour gérer étudiants, groupes et événements.
  • Utilisation de NetBeans pour le développement.

Objectifs :

  • Documenter le processus de création du site web.
  • Présenter les compétences acquises et les perspectives futures.

Description et Objectifs du Projet :

  • Développement d'une application permettant l'inscription, la connexion, et la gestion des informations sur les étudiants, groupes et événements.
  • Objectifs spécifiques :
    1. Concevoir une base de données relationnelle.
    2. Développer une interface utilisateur conviviale.
    3. Implémenter la logique métier avec des servlets et JPA.
    4. Assurer la sécurité avec un système d'authentification et d'autorisation.

Réalisation :

  • Base de Données :
    • Conception d'une base de données avec relations many-to-many entre étudiants, groupes et événements.
  • Développement de l'Application :
    1. Interface Utilisateur : Utilisation de JSP pour les pages de l'application.
    2. Logique Métier : Utilisation de servlets pour les requêtes HTTP et EJB pour les opérations CRUD.
    3. Accès aux Données : Utilisation de JPA pour la persistance et l'accès aux données.

Difficultés Rencontrées :

  • Configuration de l'Environnement :
    • Problèmes initiaux avec NetBeans et les plugins nécessaires.
  • Gestion des Dépendances :
    • Compréhension et gestion des interactions entre servlets, JSP et EJB.

Compétences Acquises :

  • Maîtrise de la programmation Java EE.
  • Développement web avancé incluant interface utilisateur, logique métier et accès aux données.
